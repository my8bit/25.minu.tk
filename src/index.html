Help: Tomatoes Work

What is Tomatoes Work?

Tomatoes Works is a free online timer that based the Pomodoro Technique® that allows you to concentrate on getting things done and being more productive and eliminate distraction. And what is Pomodoro Technique®?

It's a time management technique for breaking down your activities into 25 minutes intervals with 5 minutes break in order to stay productive and focused. You can find more about Pomodoro Technique® in this Wikipedia article.

How to use Tomatoes Work?

Choose your task on which you will be working on. Start 25 minutes timer by pressing START button. Work until the timer expires. On the first time using Tomatoes Work it requires you to give it permission to send you notifications. If You accept this permission you get the desktop (or mobile in case you use it from phone) notification. In the desktop version, you can see your progress in the title of the page as well.

What makes it unique?

There are a few unique features that make Tomatoes Works timer unique:

  • Tomatoes Work is a PWA web application, which means that you can use the same experience on your desktop browser as well as using it as a native application in your mobile phone if you install it.
  • It's connected to IoT network (Internet fo Tnihgs) through the IFTTT integration channel. This means that you can have
  • Your Pomodoro timer can be synchronized with your HipChat messenger account. When you set the timer and focused on getting things done your status is changed for Do Not Disturb which prevents you from interruptions on message notifications.And it automatically switches back to Online status when the timer expires.
  • This application is Open Source and licensed using MIT license. This means that everyone can contribute to the source repository

Do I need to do login?

In order to use the timer, you do not need to be logged in. But in case you want to sync your settings across your devices it wise to log in using GitHub or Twitter integrations.
